# Programme pour de la génétique  en Python 
# Ce que fait le programme

Le script propose 4 issues possibles :

1. **Analyse de séquence ADN** 
   - Prend un vrai fragment du génome du SARS-CoV-2 (intégré dans le code)  
   - Calcule le pourcentage GC(le pourcentage de bases Guanine (G) et Cytosine (C) dans une séquence d'ADN.)
   - Recherche un site de restriction enzymatique (EcoRI : GAATTC)  
   - Compare avec un variant fictif (compte les mutations)

2. **Modélisation d'une épidémie (modèle SIR)**  
   - Simule la propagation d'une maladie contagieuse dans une population  
   - Affiche un graphique  avec les courbes : Susceptibles, Infectés, Récupérés  
   - utile pour comprendre les épidémies comme le COVID-19

3. **Analyse d'une protéine**  
   - Étudie la séquence de l'insuline humaine  
   - compte les acides aminés les plus fréquents  
   - Affiche un histogramme de la composition

4. **Machine Learning**  
   - Utilise un exemple classique (dataset Iris) pour montrer comment un algorithme peut classer des données  
   - Analogie directe avec la classification de protéines ou la prédiction de maladies 
pour l'utiliser veuillez installer le python , ce programme a été fait sur visual studio code
